# Docs del Proyecto

Esta carpeta contiene recursos visuales para dejar el repositorio mas presentable en GitHub.

## Archivos incluidos

- `preview-template.svg`: plantilla editable para crear la imagen de portada del proyecto.
- `preview.png`: captura final recomendada (no incluida por defecto).

## Flujo recomendado (rapido)

1. Abre `preview-template.svg` con Figma, Inkscape o cualquier editor SVG.
2. Sustituye los textos de ejemplo por los de tu clase/grupo.
3. Inserta una captura real de la app si quieres un resultado mas realista.
4. Exporta como `preview.png` en esta misma carpeta (`docs/preview.png`).
5. En el `README.md` principal, descomenta o anade la imagen:

```markdown
![Preview de la app](docs/preview.png)
```

## Sugerencias visuales para clase

- Resolucion sugerida: `1280x720`.
- Mantener fondo claro y alto contraste para proyectores.
- Mostrar al menos: tabla, botones y dialogo de alta.
